# lark-agent-python

This repository contains the core code for the Gemini agent running on the VPS.
It includes the agent's main logic, built-in tools, and environment setup.

## Project Structure

- `main.py`: The main entry point for the agent.
- `tools/`: Custom scripts and utilities used by the agent.
- `venv/`: Python virtual environment.
- `requirements.txt`: Python package dependencies.
